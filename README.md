# 프로젝트명 : 뽀모도로 공부계획 웹페이지

## 기술 스택 
- **Front** : HTML, CSS, JAVASCRIPT, Jquery  
- **Back-end** : Node.js, MySQL  
- **배포** : AWS  
- **개발 기간** : 2주  

---

### 뽀모도로 공부법 소개  
뽀모도로 공부법은 학습뿐만 아니라 업무 등 다양한 작업에서 효율을 극대화하는 데 활용됩니다.  
저희 앱은 이러한 다양한 작업 유형에 맞춰 사용자가 커스터마이징할 수 있도록 설계되었습니다.

---

# 주요 기능
- ⏲️ **타이머 기능** : 직관적인 타이머 UI 제공, 뽀모도로 타이머와 일반 타이머 분리하여 사용자가 원하는 타이머로 공부 진행 가능
- 📱 **알림 기능** : 사이클이 완료되면 알림음과 화면 알림으로 휴식 또는 집중 시간 알려주기
- 📊 **통계 기능** : 사용자의 집중 시간 데이터를 기록하여 일별/주별 통계 제공, 학습 패턴 분석 및 개선 가능

---

# 페이지별 기능 소개
1. **로그인 페이지** 🔐 : 로그인 기능
2. **회원 가입 페이지** 📝 : 사용자 회원가입, 이메일 찾기, 비밀번호 바꾸기
3. **메인 페이지** 📰 : 유저 피드 작성, 수정, 삭제, 좋아요, 댓글 기능
4. **타이머 페이지** ⏳ : 뽀모도로 타이머 및 일반 타이머 기능
5. **투두 리스트 페이지** 📅 : 해야 할 일 목록 등록, 날짜별 일정 관리, 완료/미흡 설정
6. **마이 페이지** 💼 : 좋아요한 피드 목록, 일정, 공부 그래프 확인

# 개발 과정과 목표
- **아이디어** : 팀원 중 한 명이 아이디어 제공
- **UX** : 사용자 경험을 최우선으로 고려하여 간단한 인터페이스 제공
- **반응형 디자인** : 모바일과 PC 화면에 모두 적합하도록 최적화

# 부족한점 및 개선 방향
### 1. 타이머 시간 설정 ⏲️
- **현재 상태** : 타이머 시간이 고정됨
- **개선 방향** : 사용자가 직접 타이머 시간 설정 가능하도록 추가 기능 구현

### 2. 뽀모도로 소개 및 기능 설명 📚
- **현재 상태** : 로고 직관적이지 않음, 기능 설명 부족
- **개선 방향** : 첫 화면에 간략한 캐치프레이즈 및 기능 소개 추가

### 3. PC와 모바일 UI 차별화 부족 📱💻
- **현재 상태** : PC UI 불편함
- **개선 방향** : 반응형 디자인 구현하여 PC와 모바일 화면 최적화

### 4. 홈 탭 정체성 부족 💭
- **현재 상태** : 피드가 SNS처럼 보임
- **개선 방향** : 성취감을 부여하는 기능 추가 (목표 달성 인증, 통계 대시보드 등)

### 5. 버튼 색상 및 디자인 개선 🎨
- **현재 상태** : 버튼 색상 불일치
- **개선 방향** : 붉은색 계열로 버튼 색상 통일, 강조 색상 및 취소 버튼 색상 개선

### 6. 기능적 문제 ⚠️
- **로그인 문제** : 로그인 후 로그인 페이지 접근 가능
- **개선 방향** : 로그인 상태 확인 후 로그인 페이지 접근 차단, 로그아웃 기능 추가
- **투두리스트 달력** : 일정 시각적으로 표시, 타이머와 일정 연동 기능 추가

### 7. 차별점 부족 ⚡
- **현재 상태** : 기존 타이머와 차별성 부족
- **개선 방향** : 업무와 휴식을 한 화면에서 직관적으로 확인, 개인 맞춤형 타이머 추천 기능 추가

### 8. 기술적 오류 🔧
- **PC 화면 레이아웃 오류** : 타이머 시계 변경 시 레이아웃 문제
- **개선 방향** : 다양한 화면 크기에서 안정적인 레이아웃 적용
